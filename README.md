# PerfectGuide - 머신러닝 완벽 가이드

머신러닝과 데이터 분석을 위한 실습 노트북 모음입니다.

## 프로젝트 구조

- **1장**: 넘파이, 판다스 기초
- **2장**: 머신러닝 기초 및 사이킷런
- **3장**: 평가 지표
- **4장**: 분류 알고리즘 (결정트리, 앙상블, XGBoost, LightGBM 등)
- **5장**: 회귀 알고리즘
- **6장**: 차원 축소 (PCA, LDA, SVD, NMF)
- **7장**: 클러스터링 (K-Means, Mean Shift, GMM, DBSCAN)
- **8장**: 텍스트 분석
- **9장**: 추천 시스템
- **Visual**: 데이터 시각화 (Matplotlib, Seaborn)

## 설치 방법

```bash
pip install -r requirements.txt
```

## 주요 라이브러리

- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- jupyter
- xgboost
- lightgbm
- torch
